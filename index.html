Projects

Seismology and Machine Learning for the Aceh region in Indonesia

GitHub

Dec 2020 - present


Tested and retrained a recently published CNN, EQTransformer, to a new ~100 TB dataset from Aceh, Indonesia, with the goal of micro-earthquake detection, and modelling of the seismic properties of this highly active region.

Performed earthquake association and relocation, and verified the robustness of EQTransformer detections.

Developed a Python pipeline for use with a super-computing cluster, along with data processing and plotting tools.

Designing a Cell-Counting Microscope

GitHub   Project Blog

May - Jul 2021



Designed and tested a cell counting microscope, with automated motion control and focusing. Cell counting was accomplished using the OpenCV library. Wrote an accompanying web interface in Flask.

Our pipeline achieves ~90% accuracy in both synthetic and real-life tests.

Science Research Programme @ National University of Singapore

2016


Solo project to build a grating spectrometer with sub-nanometre resolution, for the purposes of characterising the spectra produced by Avalanche Photodiode flash breakdown.

Presented at the Institute of Physics, Singapore (IPS) Meeting 2017, and Singapore Science and Engineering Fair 2017.

photonLauncher

GitHub

2015-2016

An ecosystem of tools and interfaces for lab equipment. Enabled remote control and acquisition and local network management over Linux. Wrote and developed Python interfaces to perform low-level communication.

Performed second-order temporal autocorrelation measurements on photons by a Hg vapour lamp using Silicon Avalanche Photodiodes.

Presented at the Institute of Physics, Singapore (IPS) Meeting 2016.
